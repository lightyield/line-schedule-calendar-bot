# LINE スケジュール カレンダー Bot セットアップガイド

このガイドでは、LINEで転送したスケジュール文を自動で月別カレンダーに整形して返信する Bot システムの構築手順を解説します。

---

## 📋 全体フロー
1. **Google AI Studio** で Gemini API キーを取得する
2. **LINE Developers** で Messaging API の設定を行う
3. **Google スプレッドシート & GAS** の設定を行う
4. **LINE** と **GAS** を連携する

---

## 1. Google AI Studio での API キー取得
Gemini API を利用するためのキーを発行します。

1. [Google AI Studio](https://aistudio.google.com/) にアクセスし、Googleアカウントでログインします。
2. 画面上の **「Get API key」** ボタンをクリックします。
3. **「Create API key」** をクリックし、任意のプロジェクトを選択（または新規作成）して API キーを生成します。
4. 生成された API キー（`AIzaSy...` で始まる文字列）をコピーして、メモ帳などに控えておきます。
   > [!WARNING]
   > APIキーは他人に公開しないよう、厳重に管理してください。

---

## 2. LINE Developers でのチャネル作成・設定
LINEの送信・受信を制御するための Messaging API チャネルを設定します。

1. [LINE Developers Console](https://developers.line.biz/console/) にログインします。
2. **プロバイダー** を新規作成（すでに存在する場合は既存のものでも可）します。
3. **「新規チャネル作成」** から **「Messaging API」** を選択します。
4. 必要な項目（チャネル名、説明、メールアドレスなど）を入力してチャネルを作成します。
5. 作成後、以下の2つの情報を取得・確認し、控えておきます。
   - **「チャネル基本設定」** タブ： **チャネルシークレット**
   - **「Messaging API設定」** タブの最下部： **チャネルアクセストークン（長期）** （※「発行」ボタンを押して生成します）
6. **LINE公式アカウント設定の変更**
   - 同画面内の **「LINE Official Account Manager」** へのリンク、または直接管理画面を開きます。
   - **設定（右上歯車マーク） > 応答設定** を開き、以下のように設定します。
     - **応答メッセージ**: `オフ`（自動応答メッセージを無効化し、Webhook経由のみにします）
     - **Webhook**: `オン`
   - **設定 > アカウント設定 > 基本設定** を開き、以下を設定します。
     - **グループ・複数人チャットへの参加**: `許可する`（グループに招待して動作させるために必須です）

---

## 3. Google スプレッドシート & Apps Script (GAS) の設定
バッファ用スプレッドシートの作成と、GASへのプログラム配置を行います。

1. Google ドライブで **新規 Google スプレッドシート** を作成します。
2. シート名をデフォルトの `シート1` から **`buffer`** に変更します。
3. スプレッドシートのメニューから **「拡張機能」 > 「Apps Script」** をクリックします。
4. GASエディタが開いたら、デフォルトで存在する `コード.gs` ファイルを **`Code.gs`** にリネーム（または削除し新規作成）します。
5. 本プロジェクトの [src/Code.js](../src/Code.js) の中身をコピーし、GASの `Code.gs` に貼り付けます。
6. 新しくファイルを作成し、名前を **`gemini.gs`** とします（拡張子は自動的に `.gs` になります）。
7. [src/gemini.js](../src/gemini.js) の中身をコピーし、GASの `gemini.gs` に貼り付けます。
8. 画面上部の「プロジェクトを保存（フロッピーアイコン）」で保存します。

### ⚙️ スクリプトプロパティ（環境変数）の設定
GASにAPIキーなどのシークレット情報を安全に設定します。

1. GASエディタの左メニューにある **「プロジェクトの設定（歯車アイコン）」** をクリックします。
2. 画面最下部までスクロールし、**「スクリプトプロパティを追加」** をクリックして以下の情報を追加します。

| プロパティ名 | 値 | 説明 |
| :--- | :--- | :--- |
| `GEMINI_API_KEY` | `AIzaSy...` (手順1で取得したもの) | Gemini API用のアクセスキー |
| `LINE_CHANNEL_ACCESS_TOKEN` | `LINE_CHANNEL_ACCESS_TOKEN` (手順2で取得したもの) | LINEへのメッセージ返信用トークン |
| `GEMINI_MODEL` | `gemini-2.0-flash` (推奨) | 使用するGeminiモデル（未指定時は `gemini-2.0-flash` が適用されます） |

3. **「スクリプトプロパティを保存」** をクリックします。

### 🚀 ウェブアプリとしてのデプロイ
LINE Webhook からリクエストを受信するための公開URLを発行します。

1. GASエディタの右上にある **「デプロイ」 > 「新しいデプロイ」** をクリックします。
2. 種類の選択で **「ウェブアプリ」**（歯車マークから選択）を指定します。
3. 設定項目を以下のように構成します：
   - **説明**: 任意のバージョン説明（例: `v1.0.0`）
   - **次のユーザーとして実行**: **「自分」**（ご自身のGoogleアカウント）
   - **アクセスできるユーザー**: **「全員」**（LINEサーバーからのリクエストを受け付けるため必須）
4. **「デプロイ」** ボタンをクリックします。
5. （初回のみ）Googleへのアクセス権限の承認を求めるポップアップが表示された場合は、内容を確認し、アカウントを選択して **「詳細を表示」 > 「{プロジェクト名}（安全ではないページ）に移動」 > 「許可」** をクリックします。
6. デプロイ完了画面に表示される **ウェブアプリの URL** をコピーし、控えておきます。

---

## 4. LINE Webhook の接続と仕上げ
GASとLINEを接続します。

1. **LINE Developers Console** の対象チャネルの **「Messaging API設定」** タブを開きます。
2. **「Webhook URL」** の編集をクリックし、手順3で取得した **GASのウェブアプリURL** を貼り付けます。
3. **「更新」** をクリックします。
4. **「検証」** ボタンをクリックし、**「成功」** と表示されることを確認します。
   > [!NOTE]
   > もし検証でエラーが出る場合は、GASのデプロイ時の「アクセスできるユーザー」が「全員」になっているか、スクリプトプロパティに誤りがないか再度確認してください。
5. **「Webhookの利用」** を **オン** にします。

---

## 🧪 動作テスト手順
正しく動作するかテストします。

1. LINE Developers の「Messaging API設定」タブにある QRコード をスキャンし、Bot（公式アカウント）を友だち追加します。
2. Bot と自分を含む **グループチャット** を作成するか、または既存のグループチャットに Bot を招待します。
3. グループチャットに、適当なスケジュール文章（例: スクールレッスン日程などの複数メッセージ）を **一括転送** または **短時間で連続送信** します。
4. 最後の送信から約30秒〜1分ほど待つと、自動でメッセージが1つに集約され、Geminiによって整形されたマークダウン形式の月別カレンダーがグループ内に返信されます。
